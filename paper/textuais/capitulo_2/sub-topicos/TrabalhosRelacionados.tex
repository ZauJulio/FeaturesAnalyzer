\section{Trabalhos Relacionados}
    \label{sec:trabalhos-relacionados}

    O principal trabalho relacionado ao presente estudo é o \cite{SmartEnergy}. O projeto desenvolvido em 2020 tinha como objetivo a análise de dados de consumo energético, dados providos pela Universidade Federal de Campina Grande - PB. A metodologia do trabalho envolveu o teste de diversos algoritmos de regressão, clusterização e classificação, logo a plataforma proposta aqui seria um agente facilitador considerável no desenvolvimento do projeto \cite{SmartEnergy}.

    Outros dois trabalhos que se relacionam a este estudo são o \cite{brandstetter2021kmap} e \cite{pataky2019mwarp1d}, ambos tratam de plataformas utilizadas para análise e visualização de dados, utilizando a linguagem de programação Python. O primeiro trabalho apresenta uma plataforma para simulação e análise de dados em tomografia por fotoemissão. Já o segundo trata de \textit{data warping}, distorção de dados para alinhamento de caracteristicas homólogas, de dados unidimensionais em Python e \textit{PyQt}.

    A plataforma proposta neste trabalho se relaciona com ambos em linguagem, bibliotecas e contexto, pois ambos os trabalhos utilizaram Python para criar ferramentas que auxiliam na análise de dados. Entretanto, a plataforma proposta aqui se diferencia dos trabalhos citados por visar a extensabilidade e a facilidade de uso para diferentes contextos, problemas e algoritmos.
