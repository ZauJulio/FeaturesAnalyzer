\section{Contribuições}
    \label{sec:contribuicoes}

    O projeto desenvolvido neste projeto necessitou de uma extensa pesquisa bibliográfica, para ser possível identificar as melhores ferramentas e técnicas para a implementação da plataforma. O projeto considerou uma série de possibilidades de extensão e com foco na manutenção continua, não somente pelos desenvolvedores, mas também através da comunidade de desenvolvedores Python. Embora a arquitetura não esteja definida nesta etapa do trabalho, o projeto deverá seguir um conjunto de padrões de projeto e boas práticas de programação, para poder ser facilmente mantido e evoluído.

    Logo, o projeto contribuiu como um estudo de caso do desenvolvimento do projeto de uma plataforma de análise de dados, que pode ser utilizada como referência para plataformas com finalidades específicas. Além disso, o projeto contribuiu com a comunidade de desenvolvedores Python, com a proposta de uso do conjunto de interfaces e bibliotecas apresentadas anteriormente.
