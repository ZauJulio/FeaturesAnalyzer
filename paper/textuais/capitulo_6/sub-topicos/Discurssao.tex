\section{Discussão}
    \label{sec:discussao}

    O presente trabalho apresentou uma proposta de uma plataforma de análise e visualização de dados a ser desenvolvida em Python, atualmente uma das linguagens de programação mais utilizadas no mundo. A plataforma tem por objetivo ser facilmente extensível, permitindo a adição de novos algoritmos e etapas no processo de análise de dados, além de uma interface que possa ser personalizada e modificada de forma simples.

    A plataforma teve os requisitos levantados junto a um grupo de pesquisadores da área de Ciência de Dados e Processamento de sinais, que participaram do projeto \textit{SmartEnergy}, esse sendo a principal inspiração para este projeto. A plataforma do \textit{SmartEnergy} foi desenvolvida em Python, durante o projeto, mas não possuía os mesmos requisitos que os levantados aqui. Além disso, a plataforma já existente não foi projetada seguindo padrões de projeto, o que dificultou a extensão e a manutenção do código.

    Logo, a plataforma proposta no presente trabalho foi pensada para atender aos requisitos levantados, suficiente para atender as necessidades do projeto \textit{SmartEnergy}, e também para ser utilizada em outros projetos com problemas distintos.
