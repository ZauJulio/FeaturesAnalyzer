\section{Limitações}
    \label{sec:limintacoes}

    O trabalho apresentado possui algumas limitações, questões naturais deste tipo de projeto, como a alta variedade de tipos de dados, algoritmos e possíveis problemas que a plataforma poderá auxiliar na resolução. As limitações estão dispostas a seguir.

    Devido à grande variedade de algoritmos, é possível que no começo do desenvolvimento do projeto, a integração com alguns algoritmos não seja simples de tal modo como o esperado para o \textit{scikit-learn}.

    Outro ponto é que o processo de análise em \textit{pipeline}, a organização da aplicação dos algoritmos, não será simples, devido à grande variedade de interfaces que cada algoritmo e problema dispõe. Entretanto, é possível que durante a próxima etapa do projeto, seja possível identificar uma forma de organizar os algoritmos conforme as necessidades de cada problema abordado pelos dados.

    Além disso, é possível que o desenvolvimento do projeto não seja tão simples quanto o esperado, pois o mesmo é um projeto de grande porte, que envolve diversas tecnologias e ferramentas. De todo modo, o projeto seguira o planejamento estabelecido, e será desenvolvido consoante os requisitos levantados.
