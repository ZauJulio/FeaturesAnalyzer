\section{Objetivos}
    \label{sec:objetivos}

    \subsection{Objetivo Geral}
        \label{subsec:objetivo-geral}
    
    O objetivo deste trabalho é o estudo e desenvolvimento de um projeto de software com a finalidade de monitoramento de consumo elétrico através de um sistema de informação destinado a pequenas e médias empresas.   
    
    \subsection{Objetivos Específicos}
        \label{subsec:objetivos-especificos}
        
        \begin{itemize}
            \item Identificar as necessidades de gerenciamento e monitoramento energético de uma empresa. 
            \item Modelar um sistema de informação que atenda as necessidades identificadas, utilizando os conceitos de ES.
            \item Implementar o sistema modelado.
            \item Testar e validar o sistema para com aplicação em laboratório.
            \item Documentar o sistema de informação para futuras modificações.
        \end{itemize}
