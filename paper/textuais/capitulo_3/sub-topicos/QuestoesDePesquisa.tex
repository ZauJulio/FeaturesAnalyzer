\section{Questões de Pesquisa}
    \label{sec:questoes-de-pesquisa}

    A primeira questão de pesquisa é: \textit{Quais as principais funcionalidades requeridas para realizar a análise de dados?}. Esta questão foi levantada para conseuir identificar todos os requisitos necessários para o desenvolvimento do software proposto. A segunda questão é \textit{Como desenvolver o software que atenda aos requisitos levantados?} e a terceira questão é \textit{Como realizar a avaliação do software?}.

    Para responder a primeira questão de pesquisa, foi realizada uma pesquisa de campo com os pesquisadores da área de Ciência de Dados e Processamento de sinais. A pesquisa realziada envolveu uma entrevista com a finalidade de identificar as principais funcionalidades buscadas por estes pesquisadores.

    A segunda questão de pesquisa foi respondida com o auxilio da literatura, através da pesquisa bibliográfica e do emprego da engenharia de software a partir dos requisitos previamente levantados.

    Já a terceira questão foi respondida com o auxilio dos pesquisadores envolvidos na pesquisa de campo, que avaliaram o software desenvolvido e proporam testes e melhorias para o mesmo.
