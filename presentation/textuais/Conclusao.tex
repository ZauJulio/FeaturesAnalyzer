\section{Conclusão}
\label{sec:conclusao}

\begin{frame}{Conclusão}
	\quad O presente trabalho apresentou uma proposta de uma plataforma de análise e visualização de dados a ser desenvolvida em Python, atualmente uma das linguagens de programação mais utilizadas no mundo. A plataforma tem por objetivo ser facilmente extensível, permitindo a adição de novos algoritmos e etapas no processo de análise de dados, além de uma interface que possa ser personalizada e modificada de forma simples.
\end{frame}
